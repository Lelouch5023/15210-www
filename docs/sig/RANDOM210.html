Overview

The RANDOM210 signature defines an abstract type rand and provides functions to generate random values of varying types.

Interface

type rand

val fromInt : int -> rand
val next : rand -> rand
val split : rand -> (rand * rand)
val randomInt : rand -> ((int * int) option) -> int -> int
val randomReal : rand -> ((real * real) option) -> real -> real
val randomIntSeq : rand -> ((int * int) option) -> int -> int Seq.seq
val randomRealSeq : rand -> ((real * real) option) -> real -> real Seq.seq
val next : rand -> int -> int
val next : rand -> string -> int

Types

type rand
The abstract type as described in the overview.

Values

val fromInt : int -> rand
(fromInt x) evaluates to a value of type rand which uses $x$ as a seed value.
val next : rand -> rand
(next r) takes a value, $r$ of type rand and evaluates to a value of type rand.
val split : rand -> (rand * rand)
(next r) takes a value, $r$ of type rand and evaluates to a value of type rand.
